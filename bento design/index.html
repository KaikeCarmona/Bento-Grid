Oque é Bento Grid?

O Bento Grid é um sistema de layout CSS que utiliza o conceito de grids para criar layouts flexíveis e responsivos em páginas da web. Ele é inspirado na técnica de design japonês chamada "bento box", que organiza elementos em um layout semelhante a uma caixa de refeição.

Principais Características

  • Simplicidade: O Bento Grid é fácil de entender e implementar.
  • Flexibilidade: Ele permite criar layouts complexos com facilidade.
  • Responsividade: Os layouts criados com o Bento Grid são automaticamente responsivos, se adaptando a diferentes tamanhos de tela.
  • Baseado em Grids: O Bento Grid utiliza um sistema de grade flexível para posicionar elementos na página.

Como Usar o Bento Grid?

Para utilizar o Bento Grid em seu projeto, basta incluir o arquivo CSS do Bento Grid em seu HTML e começar a definir as classes de grid em seus elementos HTML. Você pode definir o número de colunas, o espaçamento entre as colunas e as linhas, além de posicionar os elementos usando classes predefinidas.

Exemplo de como se usar

1 - Definimos o elemento pai, que ira abraçar todos os outros elementos. Ele recebera a propriedade: "display: grid;"

.container {
display: grid; /*Transforma o elemento em uma Grid*/
grid-template-columns: repeat(3, 1fr); /* Define a quantidade de colunas e a largura das colunas*/
gap: 20px; /* Espaçamento entre os itens da Grid */
}

Depois passamos a quantidade de linhas e colunas que cada item ira ocupar dentro do elemento pai:

Elemento 1

.item1 {
background-color: #f0f0f0;
padding: 20px;
border: 1px solid #ccc;
text-align: center;
grid-row: span 2; /* Ocupa duas linhas */
}

Elemento 2

.item2 {
background-color: #d0d0d0;
padding: 20px;
border: 1px solid #ccc;
text-align: center;
grid-column: span 2; /* Ocupa duas colunas */
grid-row: span 2; /* Ocupa duas linhas */
}

Elemento 3

.item3 {
background-color: #b0b0b0;
padding: 20px;
border: 1px solid #ccc;
text-align: center;
grid-column: span 3; /* Ocupa três colunas */
}
